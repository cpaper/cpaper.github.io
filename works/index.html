CutClip (iOS and iPad app)

CutClip

CutClip can help you to collect and save web pages whenever you are surfing the internet in Safari. You can share, edit the offline pages whenever you like.

Features

  • Save and share web pages from Safari for offline reading
  • Store and sync data on iCloud after making an in-app purchase
  • Favorites List
  • Share contents
  • Support dark mode

X JSON Viewer (Mac catalyst app)

X JSON Viewer

a versatile and handy JSON Viewer & Formatter

Features

  • Load JSON data in versatile ways: user inputs, dragging a file or request a API
  • Pretty syntax highlight
  • Tree view
  • Level view
  • Clearly show child count for object and array nodes
  • Save and reload data from & to history list
  • Copy or Export JSON data
